# Первая лабораторная работа по JavaScript
## 1. Описание лабораторной работы
Данная работа посвящена изучению основ языка программирования JavaScript. В ходе работы были освоены навыки запуска кода в консоли браузера, создание HTML-страниц с внедренным и внешним JS-кодом, а также работа с базовыми типами данных, условиями и циклами.

## 2. Инструкции по запуску проекта
**Для запуска проекта нужно:**
1. Создать файлы `index.html` и `script.js`.
2. Поместить их в одну папку.
3. Открыть файл `index.html`.
4. Для просмотра результатов работы кода:
   Нажать `F12` или `Ctrl+Shift+I`, чтобы открыть инструменты разработчика.
   Перейдите на вкладку Console (Консоль).

## 3. Краткая документация к проекту
**Проект состоит из двух основных файлов:**
`index.html` — HTML-страница, к которой подключен скрипт. `script.js` — файл с  JavaScript.

**Основные функции кода:**
1. Вывод приветственных сообщений через `alert()` и `console.log()`.
2. Хранение данных пользователя в переменных (`name`, `birthYear`, `isStudent`).
3. Проверка условий (оценка баллов) с помощью конструкции `if и else`.
4. Демонстрация работы цикла `for` (вывод итераций от 1 до 5).

## 5. Ответы на контрольные вопросы
**1. Чем отличается var от let и const?**
* `var` — старый способ объявления. Имеет функциональную область видимости и может быть объявлен повторно.
* `let` — современный стандарт. Имеет блочную область видимости (ограничен фигурными скобками `{}`).
* `const` — как и `let`, имеет блочную область видимости, но значение переменной нельзя изменить после инициализации.

**2. Что такое неявное преобразование типов в JavaScript?**
* Это процесс, когда JavaScript автоматически приводит значение одного типа к другому для выполнения операции. Например, при сложении числа и строки `5 + "5"`, число 5 станет строкой и результатом будет `"55"`.

**3. Как работает оператор `==` в сравнении с `===`?**
* `==` (нестрогое равенство) сравнивает значения, выполняя приведение типов (например, `5 == "5"` вернет `true`).
* `===` (строгое равенство) сравнивает и значения, и типы данных без их преобразования (например, `5 === "5"` вернет `false`).
